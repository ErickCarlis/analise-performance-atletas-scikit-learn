# Análise de Performance e Fisiologia de Atletas 

Este projeto utiliza o dataset **Linnerud** do Scikit-Learn para explorar a relação entre exercícios físicos (abdominais, saltos, etc.) e indicadores fisiológicos.

##  Resumo do Projeto
O objetivo principal foi aplicar um modelo de **Regressão Linear** para entender como a performance física pode prever o peso de um atleta, um conceito fundamental em *Sports Analytics* para gestão de carga e prevenção de lesões.

##  Ferramentas Utilizadas
- Python
- Pandas & Numpy
- Scikit-Learn (Datasets, LinearRegression, Metrics)
- Matplotlib & Seaborn (Visualização de Dados)

##  Principais Insights
- Identificação de correlação forte entre medida de cintura e peso (0.87).
- O modelo apresentou um **MAE de 20.60**, evidenciando a necessidade de mais variáveis para uma precisão médica, mas validando o fluxo de dados.
